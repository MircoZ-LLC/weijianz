# weijianz.com

Personal portfolio and blog built with [Hugo](https://gohugo.io/) + [Blowfish](https://blowfish.page/) theme.

## Live Site

[weijianz.com](https://weijianz.com)

## Stack

- Hugo static site generator
- Blowfish theme (neon color scheme)
- GitHub Pages hosting
- Weglot for multilingual support

## Local Development

```bash
hugo server -D
```

## Deploy

Push to `main` — GitHub Actions auto-builds and deploys to GitHub Pages.
